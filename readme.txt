A simple and lightweight plugin that lets visitors to decide whether a story is hot or not.


== Description ==

= Main Plugin Features =

* Zuzu Hot or Not plugin adds responsive poll hot/not buttons to your posts.
* You can change the text in whatever you want like fresh/old, like/dislike, and so on.
* Zuzu Hot or Not plugin increase your blog engagement as long as you have hot content to serve them!
* Great user experience on both mobile & desktop.
* Zuzu Hot or Not plugin remembers the action of every user to avoid spamming on your blog.
* A fun way to activate your readers


== Installation ==

1. Install the plugin through the WordPress plugins screen directly (Plugins - Add new - Search for "Zuzu Hot or Not") or upload the plugin files to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' screen in WordPress.

When Zuzu Hot or Not plugin is activated, the hot/not buttons are shown on each post by default.

In the settings page you have few options:

1. Show buttons by default or Don't show buttons by default. In each case you can override the global setting in the edit post view.
2. You can also change the hot/or/not words in whatever you want, according to your language.


**Short code**

You can always use short code `[zuzu_hot_not]` in the edit post screen. If you want to add Zuzu Hot or Not plugin to your template, use `if (function_exists('zuzu_hot_not')) { zuzu_hot_not() }`.



== Frequently Asked Questions ==

= Is it free? =

Yes, absolutely!

= Can I use different background images? =

Yes. You can replace/override default background images via plugin directory/assets/img. Just keep the same image names and png format.

= Can I use short code? =
Yes!

= Does it work on multisites? =

Yes!

= Does it work with my theme? =

Yes, we've tested the plugin on various themes.

= Which browsers are supported? =

All recent browsers are supported. Internet Explorer is supported from version IE9 onwards.

= The hot/not buttons are visible on every post, can I choose on which posts they are shown? =

Yes, you can override the global setting in the edit post view.

= My buttons don't show when I reload the page. What's up? =
Some WordPress installation cache the buttons values for a while. Your buttons values are saved and will show in a couple of minutes usually.
